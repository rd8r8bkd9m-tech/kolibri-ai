# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-12

### Added
- Initial release of Kolibri AI system
- Decimal cognition layer with 0-9 impulse transduction
- Formula evolution engine with genetic algorithm
- Fractal hierarchy memory structure (10-nested digits)
- Cloud storage API with DECIMAL10X compression
- React web app with storage management UI
- Python wrapper for high-level API
- WebAssembly compilation support
- JWT authentication
- Compression statistics and analytics

### Features
- 70-80% compression ratio for text files
- 100MB file upload limit
- 10GB storage quota per user
- Real-time compression UI
- REST API with 8 endpoints
- Multi-agent voting consensus
- Digital genome blockchain
- Interpretable AI (formulas vs. neural weights)

### Performance
- Core engine: ~2MB (C executable)
- WASM module: ~500KB (browser)
- Pattern analysis: Optimized to <64 patterns
- Memory efficient: Minimal garbage collection
